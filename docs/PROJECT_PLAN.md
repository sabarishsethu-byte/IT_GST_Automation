# Project Plan

## Vision

Create a service business platform for tax filing, GST compliance, company accounting, finance automation, and MIS reporting.

The system should help the business collect leads, onboard clients, store documents, manage due dates, automate reminders, and track work from lead to delivery.

## Target Users

### Business Owner / Admin

Needs to:

- See all leads and clients
- Track upcoming due dates
- Know which documents are missing
- Assign work to staff
- Upload documents manually
- Track filing status
- Track finance automation projects
- Follow up with leads and clients

### Staff

Needs to:

- See assigned clients and tasks
- Add call notes
- Upload files
- Update filing status
- Mark documents received
- Track project stages

### Client

Needs to:

- Request a call easily
- Submit details if comfortable
- Upload documents
- See required documents
- See filing status
- Receive due date reminders
- Download acknowledgements or completed reports

## Client Folder Structure

Each client should have a logical folder inside the app:

- Profile
- KYC
- Income Tax
- GST
- Accounting
- Bank Statements
- Filed Returns
- Acknowledgements
- MIS Reports
- Automation Projects
- Internal Notes

Physical file storage can be local during MVP and moved to cloud storage later.

## Security Notes

This platform will handle sensitive personal, tax, and company finance data.

Required from early stages:

- Admin authentication
- Role-based access later
- Secure password hashing
- No plain-text storage of tax portal passwords
- Consent checkbox on forms
- Privacy Policy
- Terms of Service
- File access controls
- Audit/activity logs for important actions
- Regular backups
